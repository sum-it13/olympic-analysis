A web application to analyse and visualise data of all players and countries that have participated in the Summer Olympics throughout the history, through plots and heatmaps using data manipulation on datasets. Statistics are displayed on the basis of inputs from the user.
For development : Python, Pandas and Streamlit. For plotting : Seaborn and Plotly libraries. 

link to website : https://sum-it13-olympic-analysis-app-ffvl3z.streamlit.app/
